## 2) 相关方登记册 + 权力-利益方格

### 2.1 登记册
| 姓名 | 单位/角色 | 期望/诉求 | 态度 | 权力(1-10) | 利益(1-10) | 象限 | 管理策略 | 沟通节奏 |
|---|---|---|---|---:|---:|---|---|---|
|  |  |  | 支持/中立/反对 |  |  |  |  |  |

### 2.2 象限策略速查
- 高权力高利益：持续管理，争取成为支持者（高频同步、关键决策共创）
- 高权力低利益：谨慎管理，避免成为反对者（定期汇报、提前预警）
- 低权力低利益：少投入，监控即可（低频通报）
- 低权力高利益：多倾听，提供信息与反馈通道（需求/痛点收集）
