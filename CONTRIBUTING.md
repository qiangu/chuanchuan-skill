# 贡献指南

感谢你对 chuanchuan-skill 的兴趣！我们欢迎各种形式的贡献。

## 如何贡献

### 报告问题
- 使用 GitHub Issues 报告 bug 或提出功能建议
- 描述问题时请提供复现步骤和环境信息

### 提交代码
1. Fork 本仓库
2. 创建你的分支 (`git checkout -b feature/amazing-feature`)
3. 提交更改 (`git commit -m 'Add amazing feature'`)
4. 推送到分支 (`git push origin feature/amazing-feature`)
5. 创建 Pull Request

### 开发规范
- 保持 skill 的独立性，每个 skill 一个目录
- 在 SKILL.md 中清晰描述功能和使用方法
- 添加必要的 references 和 scripts
- 更新 README.md 中的技能列表

## 行为准则

- 尊重他人，保持友善和建设性的交流
- 接受不同的观点和经验
- 关注社区的最佳利益

## 许可证

通过贡献代码，你同意将你的贡献在 MIT 许可证下发布。
